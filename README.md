# PyQuadFilter

Implementation of digital bi-quad filter in Python

reference:
https://webaudio.github.io/Audio-EQ-Cookbook/audio-eq-cookbook.html


# Quick Start


```
pip install git+https://github.com/kurene/pyquadfilter
```
